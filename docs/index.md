# NAuTILUS  
**Neural Atlas for Terpene Interaction & Large-Scale Unified Screening**

A GPU-accelerated, graph-integrated polypharmacology inference engine for terpenes — predicting and ranking terpene–protein interactions across the human pocketome, with docking validation and Terpedia/Neptune integration.
